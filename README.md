##Finance Analytics Dashboard using Power BI

##Project Overview

This project is an end-to-end Finance Analytics Dashboard built using Power BI to analyze banking transactions and customer behavior.The dashboard helps stakeholders monitor transaction performance, customer segments, fees, taxes, and year-over-year growth trends through interactive visualizations and drill-through capabilities.

##Business Problem

Banks process thousands of transactions every day. Management requires a centralized dashboard to:
      Monitor transaction volume
      Analyze transaction amount trends
      Track fee and tax collections
      Identify top-performing customer segments
      Analyze state-wise performance
      Investigate individual transaction records

##Specific Month Analysis

  Users can select a particular month and analyze transaction performance for that period.
  
  This enables deeper trend analysis and monthly performance monitoring.

##Drill Through Analysis

  Power BI Drill Through functionality allows users to navigate from summary-level insights to transaction-level details.

For example:

Select February 2026
Drill Through
View all transactions related only to February 2026
This helps business teams investigate transaction details efficiently.

##Business Insights

Customer Insights
  Retail customers contribute the highest transaction volume.
  Premium customers generate significant transaction value.
Transaction Insights
  Successful transactions dominate overall activity.
  Failed and Pending transactions can be monitored for operational improvements.
Revenue Insights
  Fees and Tax collections contribute additional revenue streams.
  YoY analysis highlights business growth trends.
